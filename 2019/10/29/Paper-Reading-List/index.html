SSAP: Single-Shot Instance Segmentation With Affinity Pyramid

date: 2019/10/29

categories: instance segmentation

summary: This work has proposed a single-shot proposal-free instance segmentation method, which requires only one single pass to generate instances. Our method is based on a novel affinity pyramid to distinguish instances, which can
be jointly learned with the pixel-level semantic class labels using a single backbone network.

date: 2019/10/30

categories: deep stereo

summary: The proposed online unsupervised fine-tuning approach
